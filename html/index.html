O que é o "HTML"?

HTML significa HyperText Markup Language, ou seja, NÃO é uma linguagem de programação, é uma linguagem de marcação que geralmente é usada para fazer o front-end de sites junto com o CSS, ela e a lógica de programação são essenciais para aprender a programar.

Se restou alguma dúvida, existe esse artigo explicando muito melhor o que é o HTML.

Como usar o HTML?

  1. Baixe o VScode ou seu programa de edição de texto (Exemplo: sublime e atom.)
  2. Instale extensões que você achar necessárias (Exemplo: Live Server.)
  3. Quando for usar o HTML use a extensão ".html"
  4. Aproveite seu novo conhecimento!

Tags EXTRAMAMENTE úteis:

  • "a" para links.
  • "img" para imagens.
  • "p" para parágrafos.
